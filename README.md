# TaskManager

A command line task manager written in TypeScript

# Quick setup

Clone this repository

```
$ https://github.com/knaiskes/TaskManager.git
```

Install project dependencies

```
$ cd TaskManager/
$ npm install
```

Compile and run the project

```
$  tsc && node dist/index.js
```
